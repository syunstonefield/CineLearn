## 2. 概要（Summary / 132文字以内）

> Netflix・Disney+・YouTube などの字幕の単語をクリックするだけで単語帳に保存。忘却曲線アルゴリズムで最適タイミングに復習し、ドラマの没入感を保ったまま実践英語が身につきます。

（※ 半角換算でなく全角も1文字カウント。上記は約85文字で余裕あり）

## 3. 詳細説明（Detailed description）

```
CineLearn は「ドラマを観る時間」を、そのまま最強の英語学習ルーティンに変える学習支援ツールです。

■ こんな人へ
・TOEIC 900点の壁の先、"生きた英語" を身につけたい
・洋画・海外ドラマは観るけど、単語がなかなか定着しない
・学習のために動画を止めると没入感が消えて続かない

■ できること
1.【字幕の単語をワンクリック保存】
   視聴中に字幕の知らない単語をクリックするだけで、あなたの単語帳に保存されます。
   動画を止めてメモする必要はありません。

2.【AIが重要単語を自動抽出（予習）】
   エピソードの重要語だけを事前に抽出。基礎単語は自動スキップし、
   本当に覚えるべき語に絞って予習できるので、視聴中はドラマに集中できます。

3.【忘却曲線アルゴリズムで復習】
   間隔反復法（SM-2）に基づき、「忘れかけたタイミング」で復習を提案。
   「知ってた／うろ覚え／知らなかった」の3段階採点で定着させます。

4.【字幕と連動した単語カード】
   保存した単語は、実際のセリフ例文・出典（作品名・話数）付きで復習できます。
   「この単語、あのシーンで出てきた」という文脈ごと記憶に残ります。

■ 対応サービス
Netflix / Disney+ / Amazon Prime Video / YouTube

■ 料金
無料でご利用いただけます。

■ ご利用にあたって
・CineLearn は Netflix・Amazon・Disney・TMDB・OpenSubtitles と提携・公認関係にありません。
・字幕の著作権は各権利者に帰属します。本ツールは学習目的で字幕の単語・短い例文を引用・表示します（出典を明示）。
・DRM の解除・動画やコンテンツのダウンロード・再配布は一切行いません。
・例文は OpenSubtitles のデータを出典明示のうえ利用しています。

■ プライバシー
収集する情報と利用目的はプライバシーポリシーに明記しています。
個人情報を販売することはありません。
https://cinelearn-next.vercel.app/privacy

お問い合わせ・削除依頼: cinelearn.202606@gmail.com
```

## 4. Single purpose（単一目的の説明）

> CineLearn の単一目的は「対応する動画配信サービスの字幕から、ユーザーが選んだ英単語を学習用に保存し、間隔反復で復習できるようにすること」です。すべての権限はこの語彙学習という単一目的のためだけに使用します。

## 5. 権限の justification（1件ずつ根拠を書く欄）

| 権限 | 申請ダッシュボードに書く根拠 |
|---|---|
| `storage` | 保存した単語帳・学習設定・拡張のON/OFF状態・ログインセッションを端末内に保存するため。ユーザーの学習データの永続化に不可欠。 |
| `tabs` | **（削除推奨。残す場合の根拠）** 拡張アイコンのクリック時に学習アプリのページを新規タブで開くため。※ 実際には `chrome.tabs.create` は本権限なしで動くため、削除して申請するのが望ましい。 |

### host_permissions の根拠

| ホスト | 根拠 |
|---|---|
| `https://www.netflix.com/*`<br>`https://www.disneyplus.com/*`<br>`https://www.primevideo.com/*`<br>`https://www.amazon.co.jp/*`<br>`https://www.youtube.com/*` | 視聴画面に表示されている字幕を読み取り、ユーザーがクリックした単語を取得・保存し、字幕コントロールUIを表示するため。動画・音声・コンテンツ本体の取得やダウンロードは行わず、画面に出ている字幕テキストのみを対象とする。 |
| `https://cinelearn-next.vercel.app/*` | 単語の日本語訳（/api/translate）と、例文の後埋め（/api/example）を取得するための自社APIとの通信。 |
| `https://*.supabase.co/*` | ログインユーザーの単語帳をクラウド同期するため（任意・プロフィール作成時のみ）。 |
