## 2. 概要（Summary / 132文字以内）

> Netflix・Prime Video・Disney+の字幕の単語をワンクリックで単語帳に保存。実際のセリフを例文に、間隔反復で忘れる前に復習。ドラマの没入感を保ったまま、観る時間がそのまま学習になります。

（約95文字。⚠YouTube検証OKなら「Disney+」の後に「・YouTube」を追加＝+6文字で収まる）

## 3. 詳細説明（Detailed description）

```
CineLearn は「ドラマを観る時間」を、そのまま英語学習のルーティンに変える学習支援ツールです。

■ こんな人へ
・洋画・海外ドラマは観るけど、単語がなかなか定着しない
・学習のために動画を止めると没入感が消えて続かない
・単語帳の例文より、「あのシーンのあのセリフ」で覚えたい

■ できること
1.【字幕の単語をワンクリック保存】
   視聴中に字幕の知らない単語をクリックするだけで、あなたの単語帳に保存されます。
   動画を止めてメモする必要はありません。

2.【AIが重要単語を自動抽出（予習）】
   エピソードの重要語だけを事前に抽出。基礎単語は自動スキップし、
   本当に覚えるべき語に絞って予習できるので、視聴中はドラマに集中できます。
   （予習・復習は連携する学習アプリ上で行います）

3.【間隔反復（SM-2）で復習】
   「忘れる前のちょうどいいタイミング」で復習を提案。
   「知ってた／うろ覚え／知らなかった」の3段階採点で、定着を助けます。

4.【実際のセリフが例文になる単語カード】
   保存した単語には、実際のセリフ例文・出典（作品名・話数）が付きます。
   「この単語、あのシーンで出てきた」という文脈ごと思い出せます。
```

## 4. Single purpose（単一目的の説明）

> CineLearn の単一目的は「対応する動画配信サービスの字幕から、ユーザーが選んだ英単語を学習用に保存し、間隔反復で復習できるようにすること」です。すべての権限はこの語彙学習という単一目的のためだけに使用します。

## 7. スクリーンショット構成（5枚・各1280×800）

| # | 画面 | キャプション案 |
|---|---|---|
| 1 | Netflix視聴中に字幕の単語をクリック→保存ポップアップ | 「字幕の単語をワンクリックで単語帳に保存」 |

## 8. 審査で突っ込まれやすい点への備え

- **リモートコード**: content.js ほか全スクリプトは同梱のみ・外部JSのeval/injectなし（Manifest V3準拠）。
- **権限の最小化**: v1.2.1 で `tabs` 削除済み・permissions は `storage` のみ。host_permissions も機能提供ドメインに限定（YouTube は検証結果次第でさらに削減）。
- **著作権/提携**: 説明文・スクショで「Netflix公式」等と誤認させない。非提携文を説明文に含めた（§3末尾）。
- **効果の主張**: 「効果◯倍」「確実に定着」等の未裏付け表現は使わない（間隔反復・実セリフ例文など、実装事実と研究裏付けの範囲のみ）。
- **有料誘導なし**: 現状は完全無料。ストア内課金・外部決済への誘導は無し＝審査シンプル。
- **セッションブリッジ**: cinelearn-next ページの localStorage からログインセッションを拡張ストレージへ引き継ぐ設計は、自社ドメイン限定・ユーザー自身のログイン情報のみ・PPに記載済み、と説明できるようにしておく。
